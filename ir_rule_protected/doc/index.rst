=========================
 Protect ir.rule records
=========================

Installation
============

* `Install <https://flectra-development.readthedocs.io/en/latest/flectra/usage/install-module.html>`__ this module in a usual way

Usage
=====

* `Log in as SUPERUSER <https://flectra-development.readthedocs.io/en/latest/flectra/usage/login-as-superuser.html>`__
* `Activate Developer Mode <https://flectra-development.readthedocs.io/en/latest/flectra/usage/debug-mode.html>`__
* Open menu ``[[ Settings ]]>> Technical >> Security >> Record Rules``
* Select or create some record
* Click ``[Edit]``
* Set "Protected" checkbox and click ``[Save]`` button.
* RESULT: the record is protected from modifying and deleting by non-superuser.
